# Projet de tests automatises

Ce projet contient des tests automatises pour une application web de gestion de taches.

## Objectif

Apprendre a ecrire et automatiser des cas de test avec Robot Framework et Java Selenium.

## Technologies utilisees

- Robot Framework
- SeleniumLibrary
- Java + Selenium WebDriver
- Jenkins pour CI/CD

## Structure du projet

- tests/ : contient les fichiers de test Robot Framework
- ressources/ : contient les keywords reutilisables
- rapports/ : contient les rapports de tests generes
- docs/ : documentation (plan de tests, strategie)
